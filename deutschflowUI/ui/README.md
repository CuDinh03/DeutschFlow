# deutschflow UI (design export)

This folder is a **Vite/MUI bundle exported from design** ([Figma](https://www.figma.com/design/TXVarNi6E88yNVSdJnME9E/deutschflow)). It is useful for prototyping screens and parity with mocks.

## Canonical product frontend

The production app lives in **`../../frontend/`** (Next.js 14): run `npm install` and `npm run dev` there for the real DeutschFlow UX wired to the Spring Boot API.

## Running this bundle (optional)

```bash
npm i
npm run dev
```
