Thư mục này chứa bản export thiết kế v2 từ claude.ai/design. **Claude Code không mở
được link claude.ai** → bạn export ra file rồi đặt vào đây; mình đọc từ đây để restyle.

## Cách đưa vào
- Trong trang design → **Export** (hoặc copy code) từng file → lưu vào `mobile/design/v2/`.
- Thiết kế có thể là **1 file gallery** (`Native Student - Galerie.html`) chứa tất cả màn,
  hoặc **nhiều file** tách theo màn — kiểu nào cũng được, mình tự tách/đối chiếu.
- Có ảnh chụp màn kèm thì càng tốt (đặt cùng tên, vd `home.png`).
